# flet-flip-card
FletFlipCard control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-flip-card @ git+https://github.com/MyGithubAccount/flet-flip-card",
  "flet>=0.28.3",
]
```

* **PyPi dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "flet-flip-card",
  "flet>=0.28.3",
]
```

Build your app:
```
flet build macos -v
```
